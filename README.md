# ⚡️ ClickHack Studio (Carbon v4.1)

A simple and clean Markdown editor for content creators. Designed to prepare posts for platforms like Zen, VC.ru, Habr, and personal blogs.

[Russian version below / Русская версия ниже](#russian)

---

## 🛠 Features
* **Clean Copy:** Copies text as "clean" HTML. No extra spaces or broken formatting when pasting into Dzen or other CMS editors.
* **Speech Synthesis:** Listen to your text. Microsoft Edge users get high-quality neural voices.
* **Autosave:** Your drafts stay in the browser's local storage. You won't lose your work if you close the tab.
* **PWA Support:** Install it on your phone's home screen. It works full-screen without the browser address bar.

## ⌨️ Quick Markdown Guide:
* `# Title` — Header 1
* `## Subtitle` — Header 2
* `**Bold**` — Strong text
* `[Text](link)` — Hyperlink
* `- Item` — Bullet point
* `> Quote` — Blockquote

# ⚡️ КликХак Студия (Carbon v4.1)

Простой и чистый редактор для подготовки постов в Дзен, ВК, Хабр или на свои сайты.

## Основные фишки:
* **Чистое копирование:** Копирует текст как готовый HTML. В редакторах сайтов он вставляется сразу с правильными заголовками и списками без «мусора».
* **Озвучка:** Можно прослушать свой текст. В браузере Microsoft Edge доступны качественные нейронные голоса.
* **Автосохранение:** Текст хранится в памяти браузера. Если закрыть вкладку — черновик останется на месте.
* **Работает как приложение (PWA):** Если добавить на рабочий стол телефона, открывается во весь экран без лишних кнопок браузера.

## Шпаргалка:
* `# Заголовок`
* `## Подзаголовок`
* `**Жирный текст**`
* `[Текст](ссылка)`
* `- Список`
* `> Цитата`
